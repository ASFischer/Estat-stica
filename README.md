# Estatistica
Estudos sobre estatística direcionado para análise de dados e data science.

A estatística é um conjunto de ferramentas utilizadas para calcular e gerar análises de uma certa base de dados.
.Descritiva: organizar, demonstrar e resumir dados.
.Probabilidade: analisar situações sujeitas ao acaso.
.Inferencia: obter respostas sobre um fenomeno com dados representativos.

Uma análise é responsável por encontrar algum padrão dentro de uma forma resumida, ou seja, retirar uma parte do todo e caracterizar aquela população, conjunto de dados ou evento.

Amostra: é uma parte da população sendo estudada
Amostra aleatória simples: é método probabilístico de seleção de participantes ou itens de uma população para um estudo, onde cada membro tem a mesma chance igual de ser escolhido, garantindo imparcialidade e representatividade, sem viés, usando técnicas como sorteio, números aleatórios ou software, obtem se aleatoriamente. Pode ser com ou sem reposição.

Amostra estratificada: populações divididas em estratos, possuindo caracteristicas em comum.

Amostra sistematica: os elementos sao selecionados de forma que essa siga um padrao de escolha, por exemplo, a cada N pessoas uma é escolhida.

Variáveis Qualitativas descrevem qualidades ou categorias (ex: cor dos olhos, estado civil) e se dividem em Nominais (sem ordem, como sexo) e Ordinais (com ordem, como nível de escolaridade). Variáveis Quantitativas representam quantidades numéricas (ex: idade, altura) e se dividem em Discretas (contáveis, inteiros, como nº de filhos) e Contínuas (medidas, decimais, como peso ou altura)
